   - Recommended size: 200x200px
   - Format: PNG with transparent background
   - Used in: Hero section, modules, modals

   - Format: PNG with transparent background
   - Used in: Hero section, modules, modals

   - Used in: iOS devices when adding to home screen

IMAGE OPTIMIZATION TIPS:
------------------------
- Compress images to reduce file size
- Use WebP format for better compression (with fallbacks)
- Ensure images are optimized for web
- Use descriptive alt text for accessibility
- Consider using a CDN for faster loading

TOOLS FOR IMAGE CREATION:
------------------------
- Canva (for social media images)
- GIMP (free image editor)
- Photoshop (professional image editor)
- TinyPNG (for compression)
- Favicon.io (for favicon generation)

Once you have these images, simply place them in this directory and your website will be complete and ready for hosting!
